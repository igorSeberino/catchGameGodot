CATCH GAME - ALPHA JOGÁVEL FINAL
=================================

DESCRIÇÃO:
Um jogo arcade onde o jogador controla um personagem que deve coletar objetos dourados que caem do céu. O objetivo é conseguir a maior pontuação possível antes que suas 3 vidas acabem.

REQUISITOS DO SISTEMA:
- Windows 10/11 (64-bit recomendado)  
- DirectX compatível
- 2 GB RAM mínimo
- 50 MB espaço livre em disco
- Resolução mínima: 800x600

COMO INICIAR:
1. Execute "Catch Game.exe"
2. O jogo abrirá no menu principal
3. Clique em "JOGAR" para começar
4. Use "INSTRUÇÕES" para tutorial detalhado

CONTROLES PRINCIPAIS:
- A ou Seta Esquerda: Mover para esquerda
- D ou Seta Direita: Mover para direita  
- ESC: Pausar jogo / Voltar / Sair
- Mouse: Navegar e clicar nos menus

COMO JOGAR:
1. Mova o personagem verde com A/D ou setas
2. Colete círculos dourados que caem do céu (+1 ponto cada)
3. Evite que toquem o chão (perde 1 vida)
4. Você tem 3 vidas total
5. Jogo acelera conforme você progride
6. Meta: Conseguir maior pontuação possível

MENUS DISPONÍVEIS:
- Menu Principal: Jogar, Instruções, Sair
- Menu de Pausa (ESC): Continuar, Menu Principal  
- Game Over: Jogar Novamente, Menu Principal
- Tela de Instruções: Controles e regras detalhadas

FUNCIONALIDADES TESTADAS:

✓ Menu principal com navegação total
✓ Sistema de pause funcional
✓ Game over com pontuação final
✓ Controles responsivos (teclado + mouse)
✓ Progressão de dificuldade balanceada
✓ Transições entre telas estáveis
✓ ESC funciona em todos os contextos

ARQUIVOS NECESSÁRIOS:
- Catch Game.exe (executável principal)
- Catch Game.pck (dados do jogo - obrigatório)
- README.txt (este arquivo)

RESOLUÇÃO DE PROBLEMAS:
- Jogo não abre: Verifique se ambos .exe e .pck estão na mesma pasta
- Performance baixa: Feche outros programas, atualize drivers
- Travamento: Use Alt+F4, verifique DirectX instalado
- Controles não respondem: Teste teclado em outro programa
